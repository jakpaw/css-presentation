Flexible box (flexbox) layout

flexbox

Key features

  • an alternative to standard block / inline layouts
  • works horizontally or vertically
  • elements grow or shrink adapting to available space
flexbox

Vocabulary

flexbox

Flex container properties


						.flex-container {
						  display: flex | inline-flex;
						  flex-direction: row | column | row-reverse | column-reverse;
						  flex-wrap: wrap | nowrap | wrap-reverse;
						  justify-content: flex-start | flex-end | center | space-between | space-around;
						  align-items: flex-start | flex-end | center | baseline | stretch;
						  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
						}
flexbox

Flex item properties


							.flex-item {
							  flex-basis: auto | <'width'>;
							  flex-grow: <number>;
							  flex-shrink: <number>;
							  align-self: auto | flex-start | flex-end | center | baseline | stretch;
							}
...but W3C encourages to use the shorthand flex property, as it "correctly resets any unspecified components to accommodate common uses" 

							.flex-item {
							  flex: auto | none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ];
							  align-self: auto | flex-start | flex-end | center | baseline | stretch;
							}
