Flexible box (flexbox) layout

Can I Use flexbox? Data on support for the flexbox feature across the major browsers from caniuse.com.

flexbox layout

Key features

  • an alternative to standard block / inline layouts
  • works horizontally or vertically
  • elements grow or shrink adapting to available space
flexbox layout

Vocabulary

flexbox layout

Flex container properties


						.flex-container {
						  display: flex | inline-flex;
						  flex-direction: row | column | row-reverse | column-reverse;
						  flex-wrap: wrap | nowrap | wrap-reverse;
						  justify-content: flex-start | flex-end | center | space-between | space-around;
						  align-items: flex-start | flex-end | center | baseline | stretch;
						  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
						}
flexbox layout

Flex item properties


							.flex-item {
							  flex-basis: auto | <'width'>;
							  flex-grow: <number>;
							  flex-shrink: <umber>;
							  align-self: auto | flex-start | flex-end | center | baseline | stretch;
							}
...but W3C encourages to use the shorthand flex property, as it "correctly resets any unspecified components to accommodate common uses" 

							.flex-item {
							  flex: auto | none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ];
							  align-self: auto | flex-start | flex-end | center | baseline | stretch;
							}

Multi-column layout

Can I Use multicolumn? Data on support for the multicolumn feature across the major browsers from caniuse.com.

multi-column layout

Key features

  • block layout extension
  • automatically divides content into columns
  • you can specifiy column count and/or minimum column width
  • works on block elements and on text
multi-column layout

Columns container properties


						.columns-container {
						  column-count: <integer> | auto;
						  column-width: <length> | auto;
						  column-gap: <length> | <percentage> | normal;
						  column-fill: auto | balance;
						  column-rule: <'column-rule-width'> || <'column-rule-style'> || <'column-rule-color'>;
						  ...
						}
multi-column layout

Children properties


						.columns-child {
						  break-inside: auto | avoid | avoid-column | column;
						  break-before: auto | avoid | avoid-column | column;
						  break-after: auto | avoid | avoid-column | column;
						  column-span: none | all;
						}
break-before, break-after and column-span not supported by some browsers

Grid layout

Can I Use css-grid? Data on support for the css-grid feature across the major browsers from caniuse.com.

grid layout

Key features

  • "most powerful layout system available in CSS"
  • for large-scale, 2-dimensional layouts
  • like a table with rows and columns, but with many more possibilites
  • you can specify rows, columns and named areas, and place elements in them
grid layout

Grid container properties


							.grid-container {
							  display: grid | inline-grid | subgrid;
							  grid-template-columns: <track-size> ... | <line-name> <track-size> ...;
							  grid-template-rows: <track-size> ... | <line-name> <track-size> ...;
							  grid-template-areas: none | <string>+;
							  grid-column-gap: <length> | <percentage>;
							  grid-row-gap: <length> | <percentage>;
							  ...
							}
grid layout

Grid item properties


						.grid-item {
						  grid-column-start: <number> | <name> | span <number> | span <name> | auto;
						  grid-column-end: <number> | <name> | span <number> | span <name> | auto;
						  grid-row-start: <number> | <name> | span <number> | span <name> | auto;
						  grid-row-end: <number> | <name> | span <number> | span <name> | auto;
						  grid-area: <name>;
						}

Transitions

Can I Use css-transitions? Data on support for the css-transitions feature across the major browsers from caniuse.com.

transitions

Key features

  • make property changes happen over a period of time
  • intermediate states are automatically calculated
  • you can control delay, duration and easing of transition
  • not all properties can be animated
transitions

How to use


						div {
						  transition-delay: <time>#;
						  transition-duration: <time>#;
						  transition-property: none | all | <custom-ident>;
						  transition-timing-function: linear | <cubic-bezier-timing-function> | <step-timing-function> | <frames-timing-function>;
						}
... or as a shorthand property 

						div {
						  transition: [transition-property] [transition-duration] [transition-timing-function] [transition-delay];
						}
